# Portfolio - SALOHIMAMPIONONA Maminiala Toussaint

Portfolio moderne, responsive et prêt pour GitHub Pages.

## Contenu
- Hero section
- À propos
- Compétences
- Projets
- Contact
- Footer
- Animation légère au scroll
- Menu mobile

## Structure
- `index.html` : structure principale du site
- `styles.css` : mise en forme et responsive design
- `script.js` : navigation mobile et apparition au scroll

## À personnaliser
Remplacer les valeurs de démonstration suivantes :
- `votre.email@exemple.com`
- `+261 00 00 000 00`
- `github.com/votre-profil`
- les liens GitHub de chaque projet

## CV
Pour activer le bouton de téléchargement, ajoutez un fichier PDF nommé :
- `cv-maminiala-toussaint.pdf`

## Déploiement sur GitHub Pages
1. Pousser le projet sur GitHub.
2. Aller dans les paramètres du dépôt.
3. Ouvrir la section GitHub Pages.
4. Sélectionner la branche principale et la racine `/`.
5. Enregistrer.

Le site sera alors accessible via l’URL fournie par GitHub Pages.
